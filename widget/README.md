# Safe Helpline India Widget

The widget is a dependency-free JavaScript embed for displaying the verified Safe Helpline India entries on a static page.

```html
<div id="safe-helpline-widget"></div>
<script src="./safe-helpline-widget.js"></script>
```

You can also render manually:

```html
<div id="custom-helpline-widget"></div>
<script src="./safe-helpline-widget.js"></script>
<script>
  window.SafeHelplineWidget.render("#custom-helpline-widget");
</script>
```

## Privacy

- No tracking.
- No cookies.
- No local storage.
- No network requests from the widget itself.
- No contact, location, or form collection.

## Data Policy

The widget includes only the two currently verified public helpline entries: 112 ERSS and 181 Women Helpline. Each card links to its public source.
